# VerificationGuide

> SystemVerilog / UVM / SoC 设计学习指南

系统化的数字芯片验证与设计学习文档，涵盖 SV 语法基础、UVM 验证方法学、SoC 架构与小模块实战练习。

## 内容结构

| 模块 | 内容 |
|---|---|
| **SystemVerilog** | 语法基础、语法进阶、数据类型、class 示例 |
| **UVM验证** | UVM 知识详解（基于实例讲解） |
| **SoC设计** | 基本架构、总线协议、入门/进阶小模块练习 |

## 本地运行

```bash
npm install
npm run dev
```

访问 http://localhost:4321/VerificationGuide
